# AWS Static Portfolio Site
This project is part of my AWS Cloud Journey. 

## Project Overview
A basic static website hosted on AWS using high-availability services.

## Architecture
* **Storage:** Amazon S3
* **Content Delivery:** Amazon CloudFront (Planned)
* **DNS:** Amazon Route 53 (Planned)

## How to Deploy
1. Create an S3 bucket with public access.
2. Enable "Static website hosting".
3. Upload `index.html`.
